# 项目背景：TraceGuard AIGC图像安全审核平台
我是竞赛项目贺杰，负责【可解释检测取证模块+篡改可疑区域定位模块】开发，上游模块为张潇开发的跨域AIGC检测模块（MambaOut-Small去全局池化骨干+MK-MMD域自适应）。
项目文档核心要求：
1. 两大核心子模块：可解释热力图生成模块、局部篡改定位分析模块
2. 输出内容：
   1）伪影热力图（精准高亮AIGC网格、局部色偏、扩散残差痕迹，不能是无意义可视化）
   2）篡改掩膜图、可疑区域坐标框列表
   3）标准化自然语言解释文本，用于平台报告导出
   4）局部区域风险分数，供给多证据融合模块做风险分级（低/中/高风险）
3. 工程约束：
   - 底层工程基于 FakeTrace，代码存放路径 detection/
   - 需提供可调用Python类、FastAPI内部接口、批量处理脚本
   - 兼容CLI、Web UI双端调用，输出图片为PIL.Image格式，可前端渲染、写入PDF检测报告
   - 轻量化，单GPU低延迟推理，适配线上审核场景
4. 配套任务：代码附带注释、README调用示例、测试用例（真实图/完整AIGC图/局部篡改图三类样本）
