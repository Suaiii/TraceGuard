# Localization boundary evidence

`aigc_synthetic_15_summary.json` and `aigc_synthetic_15_threshold_sweep.csv` record a reproducible boundary evaluation on a Facebook-derived balanced AIGC subset. Ten synthetic tampered images have known pasted-region masks and five untouched clean images are negative controls. The result is intentionally used as a limitation: high image-level detection with very low pixel overlap and 100% clean false-positive rate does not support a pixel-accurate localization claim.
